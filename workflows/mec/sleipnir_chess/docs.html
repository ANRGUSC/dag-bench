SLEIPNIR Chess App (5 Moves)

ID: mec.sleipnir_chess  |  Domains: mec, edge-computing  |  Completeness: structure-and-costs  |  Cost Model: deterministic

Mobile chess application DAG from the SLEIPNIR offloading simulator (De Maio and Brandic, CCGrid 2018). Models 5 sequential chess moves, each consisting of 4 tasks: UI display, board update, move computation (AI), and output display. The output of each move feeds into the UI of the next move, creating a long linear chain of 20 tasks. The largest SLEIPNIR app by task count, parameterized by chessMovesNum (default 5). Interesting for stress-testing schedulers on long chains.
