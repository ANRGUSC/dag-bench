SLEIPNIR Facebook App

ID: mec.sleipnir_facebook  |  Domains: mec, edge-computing  |  Completeness: structure-and-costs  |  Cost Model: deterministic

Mobile social media application DAG from the SLEIPNIR offloading simulator (De Maio and Brandic, CCGrid 2018). The GUI fans out to authentication (GET_TOKEN) and API request in parallel, then proceeds through response processing, file upload, image filter application, and final post creation. Near-linear chain with an early fork-join at GUI, 7 tasks total.
