SLEIPNIR Antivirus App

ID: mec.sleipnir_antivirus  |  Domains: mec, edge-computing  |  Completeness: structure-and-costs  |  Cost Model: deterministic

Mobile antivirus application DAG from the SLEIPNIR offloading simulator (De Maio and Brandic, CCGrid 2018). The UI task fans out to load virus definitions and scan a file in parallel, both feeding into a comparison stage that produces the final output. Classic fork-join diamond pattern with 5 tasks.
