SLEIPNIR Face Recognizer App

ID: mec.sleipnir_facerecognizer  |  Domains: mec, edge-computing, image-processing  |  Completeness: structure-and-costs  |  Cost Model: deterministic

Mobile face recognition application DAG from the SLEIPNIR offloading simulator (De Maio and Brandic, CCGrid 2018). The UI triggers face matching, which initializes a match database and detects faces in an image, with a bypass edge from matching to detection. Chain-with-bypass pattern with 5 tasks, where the heaviest task (DETECT_FACE at 1600 MI) performs the core face detection computation.
