Pipeline-Parallel Face Analysis System

ID: edge.face_analysis_pipeline  |  Domains: edge-computing, ml-pipeline, image-processing  |  Completeness: structure-and-costs  |  Cost Model: deterministic

Edge-based face analysis pipeline with pipeline-parallel multi-task learning, faithfully extracted from Figures 4-5 of a Computers (MDPI) 2025 paper. YOLOv8 detects and crops heads, VGG-Face extracts features, then three parallel MTL classification modules (hair, gender, face ID) run on distributed Raspberry Pi edge devices, with results aggregated at a write-back stage. Each MTL module produces two classification outputs (e.g., hair color + hairstyle, gender + smile, face recognition + age estimation).
